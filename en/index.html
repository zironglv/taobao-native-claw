Features

Why Taobao Native CLI

Four core advantages for a powerful AI shopping assistant

Ready to Use

Reuses your logged-in Taobao desktop client state. No CAPTCHAs, no anti-bot challenges. Install and go.

Native Operations

Based on Taobao desktop client native input events. More stable and reliable than browser automation. Behaves like real users.

Shop Anywhere

Full features unlocked. Your assistant helps you search products, view details, manage orders, WangWang chat, and more - the complete Taobao shopping workflow.

Seamless OpenClaw Integration

Supports OpenClaw/CoPaw/Qoder and other OpenClaw-compatible AI assistants. One command to install.

FAQ

Frequently Asked Questions

Common questions about Taobao Native CLI

Is Taobao Native CLI free? +

Yes, Taobao Native CLI is completely free. Simply install the Taobao desktop client and Taobao Native Skill to get started.

Which AI assistants are supported? +

Currently supports all OpenClaw-compatible AI assistants, including OpenClaw, CoPaw, Qoder, and more. Check if your AI assistant supports OpenClaw Skill format.

Is my Taobao account safe? +

Absolutely safe. Taobao Native CLI directly controls your locally logged-in Taobao client. All operations happen on your computer - no account info or passwords are uploaded. AI only controls the client interface via CLI commands, exactly like you would manually.

Which operating systems are supported? +

Currently supports macOS (Apple Silicon and Intel) and Windows. Linux support is under development.

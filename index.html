常见问题

FAQ

关于淘宝桌面版CLAW的常见问题解答

淘宝Claw插件版是免费的吗？ +

是的，淘宝Claw插件版完全免费，你可以自由安装和使用。只需安装淘宝桌面客户端和Taobao Native Skill即可开始使用。

支持哪些 AI 助手？ +

目前支持所有兼容 OpenClaw 标准的 AI 助手，包括 OpenClaw、CoPaw、Qoder 等。如果你使用其他 AI 助手，可以查看是否支持 OpenClaw Skill 格式。

我的淘宝账号安全吗？ +

绝对安全。淘宝桌面版CLAW直接操作你本地已登录的淘宝客户端，所有操作都在你的电脑上完成，不会上传任何账号信息或密码。AI 只是通过 CLI 命令操控客户端界面，与你自己操作完全一致。

支持哪些操作系统？ +

目前支持 macOS（Apple Silicon 和 Intel）以及 Windows 系统。淘宝桌面客户端需要从官方网站下载对应版本。

能完成哪些购物操作？ +

支持大部分淘宝购物操作：商品搜索、商品详情查看、加入购物车、订单查询、催发货、开发票、旺旺聊天、收藏夹管理、浏览历史查看、图片搜索等。持续更新中...
